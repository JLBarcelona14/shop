PRODUCT

(SAVE/ADD PRODUCT)
URL: /save/product

(UPDATE PRODUCT BASE ON ID PARAMETER)
URL: /update/product/:id 

POST Body(JSON) Example:
{
	"product_name": "Sample Product",
	"category": "Sample Product Category",
}

(GET ALL PRODUCTS)
URL: /get/product

(FIND 1 RECORD FROM PRODUCT BASE ON ID PARAMETER)
URL: /find/product/:id

(DELETE PRODUCT BASE ON ID PARAMETER)
URL: /delete/product/:id
